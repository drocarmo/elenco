To-do App

Elenco at its core servers as a simple to-do web application. Opening a new tab activates Elenco, displaying current tasks with access to easily create more tasks.

Syncs Everywhere

Syncing tasks to all the Chrome browsers your google account is logged into is essential. Edit & delete tasks and it syncs everywhere.

Personalized For You

You can set your own background color from multiple given options to help keep you focused.

Open Source

Elenco was built out in the open. We're building new features and killing issues as they come. It's whatever you want it to be.
